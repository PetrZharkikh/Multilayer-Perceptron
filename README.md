# MLP from scratch

Реализация многослойного перцептрона на Python + NumPy без PyTorch.
Проект выполнен в рамках задания по теме MLP СБЕРовского курса по компиляторам: реализованы линейные слои, функция активации ReLU, функция ошибки MSELoss, а также обучение с помощью обратного распространения ошибки и градиентного спуска. :contentReference[oaicite:0]{index=0}

## Описание задания

Необходимо реализовать многослойный перцептрон, состоящий из:
- линейных слоёв
- слоёв активации ReLU
- функции ошибки MSELoss

Также требуется:
- реализовать прямой проход (`forward`)
- реализовать обратный проход (`backward`)
- обучить модель градиентным спуском
- использовать батчи
- использовать аккумуляцию градиентов
- использовать clipping градиентов
- использовать регуляризацию

Сеть должна аппроксимировать функцию:

\[
f(x_1, x_2, x_3, x_4) =
\begin{pmatrix}
x_1^2 \\
3x_2 \\
5x_4 - x_3 \\
3
\end{pmatrix}
\]

Датасет генерируется автоматически с помощью случайных входных данных из нормального распределения. :contentReference[oaicite:1]{index=1}

## Структура решения

В проекте реализованы следующие компоненты:

### `Linear`
Линейный слой:
\[
y = Wx + b
\]

Поддерживает:
- `forward`
- `backward`
- `update`
- `zero_grad`
- `clip_grad`

### `ReLU`
Функция активации:
\[
ReLU(x) = \max(0, x)
\]

Поддерживает:
- `forward`
- `backward`

### `MSELoss`
Среднеквадратичная ошибка:
\[
MSE = \frac{1}{n}\sum (y_{pred} - y_{true})^2
\]

Поддерживает:
- `forward`
- `backward`

### `MLP`
Главный класс многослойного перцептрона.  
Объединяет слои и реализует:
- `forward`
- `loss`
- `backward`
- `update`
- `zero_grad`
- `clip_grad`
